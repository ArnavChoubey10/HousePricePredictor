# House Price Predictor 

This is a very small Machine Learning project using Linear Regression.
It predicts house prices based on:
- area (sqft)
- bedrooms
- age (years)

## Project Structure

HousePricePredictor.py/
├── data/
│ └── data.csv
├── model.py
└── README.md


## How to Run
1. Activate virtual environment:
   `.\venv\Scripts\Activate`
2. Install requirements:
   `pip install pandas scikit-learn`
3. Run the model:
   `python model.py`

## Output Example
- Shows model accuracy (R²)
- Shows predicted price for a sample house

## Dataset
Tiny dataset of 7 rows for fast training (educational).
